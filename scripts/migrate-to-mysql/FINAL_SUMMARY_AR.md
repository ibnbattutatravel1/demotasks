```bash

# 2. ثبّت المكتبة

# 3. أعد MySQL

# 4. حدّث .env
# أضف معلومات MySQL

# 5. شغّل الترحيل
npm run migrate:mysql:all

# 6. حدّث client.ts
# اتبع التعليمات في START_MYSQL_MIGRATION_AR.md

# 7. اختبر
```
